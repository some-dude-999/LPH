SPANISH STAGE 3: TRANSLATION QUALITY REVIEW (250 PACKS)

THE #1 RULE: ACTUAL TRANSLATIONS, NOT DESCRIPTIONS
---------------------------------------------------
Translations must be what you'd actually SAY in that language.
NOT grammar labels. NOT descriptions. NOT parenthetical notes.

WRONG: "definite article" (grammar label)
RIGHT: "the" (actual translation)

WRONG: "(formal you)" (parenthetical note)
RIGHT: "you (formal)" (translation with annotation)

Ask yourself: "Would a native speaker ever say this word?"
If no, it's not a translation - it's a description.

COLUMN 0 (spanish): SACRED - NEVER EDIT
OTHER COLUMNS: Find ACTUAL translations

CHECKLIST:
[+] Theme - Check Pack_Title, same word = different translation per theme
[+] Natural - Everyday words, not formal/technical
[+] Native - What speakers actually say
[+] Annotations OK for disambiguation: (masculine)/(feminine)/(formal)/(informal)
[-] NO grammar labels as translations
[-] NO descriptions instead of translations
[-] NO parenthetical notes as entire translation
