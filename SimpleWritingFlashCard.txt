DATA SOURCES:
- Module: https://some-dude-999.github.io/LPH/SpanishWords/Jsmodules/act1-foundation.js
  Provides wordpacks with Spanish vocabulary and translations (english, chinese, portuguese)
- LocalStorage: Saves user preferences (voice selection, speech speed)
- Web Speech API: Text-to-speech for pronunciation

FUNCTIONALITY:
- Main Purpose: Translation practice - see English, type Spanish translation
- Auto-Start: Game automatically starts with Pack 1 and English language on page load
- Game Flow:
  1. English translation appears on front
  2. Type the Spanish translation (case and accent insensitive)
  3. Correct letters appear, wrong letters turn red
  4. Press Enter or flip card to see answer
  5. Back shows correct Spanish word
  6. Auto-pronounces Spanish word when flipped
  7. Automatically advances to next card

- Key Features:
  * English word on FRONT
  * Type Spanish translation
  * AUTO-PRONOUNCE Spanish word on flip to back (key feature!)
  * Case and accent insensitive matching (√° = a, √â = e)
  * Wrong letter penalty: Auto-adds 2 duplicate cards for practice
  * Mechanical keyboard sound for correct letters
  * Red "X" marks on wrong letters
  * Wrong attempt counter displays in corner

- Key Functions:
  * loadWordpacks() - Dynamically imports vocabulary modules and auto-starts game
  * initializeDeck(packKey) - Creates shuffled deck from selected wordpack
  * updateDisplay() - Shows English word + typing interface
  * initializeTypingDisplay() - Sets up blank word with underscores
  * handleTypingInput(key) - Processes typed letters, adds wrong letter penalties
  * normalizeChar(char) - Removes accents for matching (√°‚Üía, √©‚Üíe)
  * playKeyboardSound() - Tactile feedback for correct letters
  * addDuplicateCards(count) - Penalty for wrong attempts
